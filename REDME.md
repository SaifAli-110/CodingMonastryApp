# Project Concept
# The theme of a "Coding Monastery" focuses on minimalism, focus, and structure. The app serves as a portal where "initiates" (students) can access coding "scripts" (lessons), track their # "meditation" (coding hours), and join "temples" (study groups).

# Core Features (Bootstrap-Powered)
# Dojo Dashboard: A clean user interface using Bootstrap Cards to display enrolled courses, progress bars, and daily challenges.

# Focus-Mode Lesson Player: A layout using the Grid System where the video/content takes up 8 columns and the syllabus/notes take up 4.

# Monk Leaderboard: A Table component showing student rankings based on "Karma points" earned by solving bugs.

# Responsive Navigation: A Navbar with a toggleable "Zen Mode" (Dark Mode) switch.

# Admissions Form: A sleek Modal or styled Form for new students to sign up for specific coding "paths" (Web, AI, Mobile).

# 🛠️ Technical Breakdown (Bootstrap Components)
# 1. Layout & Grid
# Using the .container, .row, and .col classes, you can ensure the dashboard scales perfectly. For instance, a 3-column layout on desktop can stack into a single column on mobile automatically.

# 2. The "Zen" Navigation
# Sticky Top Navbar: Keeps navigation accessible while scrolling through long coding tutorials.

# Offcanvas Sidebar: Used for mobile users to access their profile and settings without cluttering the main view.

# 3. Styled Components
# Progress Bars: To show how close a student is to "Enlightenment" (course completion).

# Badges: Small colorful indicators (e.g., <span class="badge bg-warning">JavaScript Master</span>) to highlight skills.

# Accordions: Perfect for a "FAQ" section or a structured course syllabus where lessons are grouped by modules.

# 🎨 Design Palette (The "Monastery" Aesthetic)
Since you are using Bootstrap, you can override the default colors to match the theme:

# Primary Background: Off-white or light grey (#f8f9fa) to reduce eye strain.

# Accent Color: Deep Saffron or Zen Green for buttons and active states.

# Typography: Clean, sans-serif fonts (like Inter or Roboto) for high readability.
